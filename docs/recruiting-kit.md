# MasterBook — комплект для набора мастеров

Практичный набор текстов, чтобы приглашать мастеров через сарафан и WhatsApp.
Всё готово к copy-paste. Правь под свой голос.

---

## 1. Питч в одну фразу

> **MasterBook — бесплатное приложение, где мастер ведёт записи клиентов, а ещё
> получает свою страницу на сайте, куда клиенты записываются онлайн. Платишь
> только если хочешь подняться выше в каталоге — никакой комиссии с записей.**

Три бесплатных кита (повторяй их везде):
1. **Бесплатное приложение-CRM** — клиенты, записи, напоминания, доход.
2. **Бесплатная страница на сайте** — мастера видно в каталоге своего города.
3. **Онлайн-запись** — клиент жмёт «Записаться», заявка падает мастеру в приложение.

Деньги берём **только за продвижение** (выше в каталоге / метка VIP), оплата
локальным переводом. Не комиссия с записей, не подписка за софт.

## 3. Инструкция для мастера: как попасть на сайт (5 минут)

Отправляй мастеру после согласия:

1. **Установи приложение MasterBook** (ссылку даст менеджер).
2. **Заполни профиль:** имя/название, город, услуги с ценами. (Минимум для сайта:
   имя + город + хотя бы одна услуга.)
3. **Нажми «Показать на сайте»** (раздел публикации) и **зарегистрируйся** —
   все твои данные сами переедут в облако.
4. **Готово.** Твоя страница: `masterbook-app.vercel.app/m/твой-адрес`.
   Делись ссылкой в Instagram/WhatsApp — клиенты записываются прямо там.

> Город важно указать **точно** — клиент ищет мастеров по своему городу, и тебя
> увидят только местные. (Из приложения город вписывается вручную; на сайте в
> кабинете `masterbook-app.vercel.app/cabinet` можно выбрать страну и город из
> списка.)

## 4. Частые вопросы

**Это правда бесплатно?**
Да. Приложение, страница на сайте и онлайн-записи — бесплатно. Платно только
продвижение (выше в каталоге), и только если сам захочешь.

**Вы берёте процент с моих записей?**
Нет. Ноль комиссии. Записи и деньги — напрямую между тобой и клиентом.

**Как клиенты меня найдут?**
Через каталог сайта по твоему городу и категории, и по прямой ссылке, которой
ты делишься в соцсетях.

**Что если я в другом городе/стране?**
Каталог разделён по странам и городам. Тебя видят только клиенты твоего города —
мастер из Москвы не попадёт к клиенту из Душанбе, и наоборот.

**Как мне платят клиенты?**
Как договоритесь напрямую (наличные/перевод). Мы в оплату между вами не лезем.
