# MasterBook

**CRM для частных мастеров.** Записи клиентов, финансовая статистика, напоминания — всё что нужно чтобы вести дело без хаоса.

## Для кого

- Маникюр, ногтевой сервис, парикмахерская
- Репетиторы и преподаватели
- Фотографы
- Мастера по ремонту
- Любой частный специалист, работающий с клиентами

## Возможности

- Календарь записей с напоминаниями за час
- База клиентов с тегами (VIP, проблемный, новый)
- Финансовая аналитика по дням / неделям / месяцам
- Экспорт данных в JSON — данные всегда с тобой
- Face ID / Touch ID защита
- Тёмная и светлая темы

## Безопасность

- Шифрование данных при передаче (HTTPS)
- Row-Level Security — каждый мастер видит только свои данные
- Локальная защита биометрией
- Удаление аккаунта прямо из приложения
