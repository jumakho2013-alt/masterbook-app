# MasterBook — сайт-каталог (Next.js)

Витрина мастеров поверх **той же Supabase**, что и мобильное приложение. Читает
**публичный слой** (`published`-мастера + их услуги + отзывы) — приватная CRM
(клиенты/записи/финансы) анониму недоступна (RLS).

## Что готово (читает живую базу)
- **Главная** — hero, категории, лучшие мастера.
- **Каталог** (`/catalog`) — список опубликованных мастеров, поиск по имени И
  профессии, фильтр по городу, премиум-первыми.
- **Профиль мастера** (`/m/<slug>`) — услуги+цены (сомони), отзывы, рейтинг, VIP,
  SEO-метаданные (шеринг в мессенджерах), + **онлайн-запись** и WhatsApp/звонок.
- **Онлайн-запись** — форма на профиле → Edge Function `book` (service-role):
  валидирует мастера/услугу/рабочие день-часы/пересечение слотов, заводит клиента и
  запись. Запись приходит мастеру в приложение обычным синком (`updated_at`).
- **Кабинет мастера** (`/cabinet`) — вход тем же Supabase Auth, что и приложение:
  редактор публичной страницы (город/район/био/slug/контакты/публикация),
  список записей, баланс, покупка премиума (`redeem_promotion`), заявки на пополнение.
- **Админка** (`/admin`) — Edge Function `admin-payments` (проверка e-mail по
  `ADMIN_EMAILS`): список оплат на проверке, подтвердить/отклонить.
- **Тёмная тема (Noir)** — по системной настройке.

## Edge Functions (задеплоены в Supabase)
- `book` — публичная онлайн-запись (verify_jwt включён; фронт шлёт anon-ключ).
- `admin-payments` — подтверждение оплат, доступ только из `ADMIN_EMAILS`.
- Гранты: `confirm_payment` отозван у anon/authenticated (вызывает только service_role
  через `admin-payments`) — миграция `harden_confirm_payment_grants`.

## Запуск
```bash
cd web
cp .env.local.example .env.local   # вставь те же значения, что в app .env.local
#   EXPO_PUBLIC_SUPABASE_URL      → NEXT_PUBLIC_SUPABASE_URL
#   EXPO_PUBLIC_SUPABASE_ANON_KEY → NEXT_PUBLIC_SUPABASE_ANON_KEY
npm install
npm run dev        # http://localhost:3000
```
Чтобы в каталоге кто-то появился: в приложении у мастера зайди Профиль → «Моя
страница на сайте» → заполни город + укажи услуги → включи «Показывать на сайте».
